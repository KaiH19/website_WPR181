Overview of the World Wide Web

The World Wide Web (WWW) is an information space consisting of interconnected documents and resources linked by hyperlinks. It allows users to navigate and access a wide range of multimedia content using web browsers. The WWW revolutionised the way information is shared and accessed, enabling the creation of websites, online services, and communication platforms.

History of the Internet (Including Importatnt dates)

  1. 1962 : J.C.R. Licklider introduces the concept of a “Galactic Network” and envisions a globally interconnected set of computers.

  2. 1969 : The ARPANET project begins, establishing the foundation for the internet.

  3. 1971 : The fist email is sent by Ray Tomlinson.

  4. 1973 : Vint Cerf and Bob Kahn publish the first paper describing the Transmission Control Protocol, a key protocol for data transmission on the Internet.

  5. 1983 : The Domain Name System (DNS) is introduced, enabling the use of human - readable domain names.

  6. 1989 : Tim Berners-Lee proposes the concept of the "Wolrd Wide Web".

  7. 1990 : The first web browser, "WorldWideWeb", is released. And the First web server is released, by Tim Berners-Lee.

  8. 1991: The World Wide Web is made publicly accessible, marking the official birth of the modern web.

  9. 1993 : The graphical web browser Mosaic is released, popularising the web.

  10. 1994 : Netscape Navigator, a commercial web browser, is launched.

  11. 1995 : NSFNET, a major backbone of the Internet, shifts from government control to commercial providers.

  12. 1998 : Google is founded, revolutionising web search.

  13. 2000: The dot-com bubble bursts, leading to both the decline of many internet-based businesses and the eventual resurgence of a more sustainable online economy

  14. 2004 : Web 2.0 emerges, emphasising user-generated content and dynamic web applications.

  15. 2007 : Apple introduces the iPhone, ushering in the era of smartphones and mobile internet access

  16. 2010 : Instagram is launched, focusing on photo sharing, and paving the way for the growth of visual social media.

  17. 2014: The Internet of Things (IoT) gains traction, connecting everyday objects and devices to the internet.

  18. 2016 : The completion of the IPv6 transition helps address the growing shortage of available IP addresses.

WWW vs the Internet

Basis of comparison WWW Internet
Definition The internet is a global network of interconnected computer networks that use the TCP/IP protocol to connect devices worldwide The World Wide Web or WWW refers to online content that is formatted in HTML and accessible via the HTTP/HTTPS protocol.
Founded Internet was founded in the late 1960s. In 1989, Tim Berners-Lee, an English scientist, created the World Wide Web.
Architecture The internet's architecture is decentralized and follows a peer-to-peer model. It is a network of networks, meaning that it connects various autonomous systems and networks that use the TCP/IP protocol to communicate with each other. The World Wide Web is based on a client-server architecture. Clients (typically web browsers) send requests to web servers, which then respond by serving web pages and resources to the clients.
Functionality The internet is a vast network that connects computer networks all over the world. World Wide Web is a service provided by the internet.
Type The internet is primarily hardware-based. In comparison to the Internet, WWW is more software-oriented.
Protocol The Internet utilizes the Transmission Control Protocol/Internet Protocol (TCP/IP). The World Wide Web utilizes the Hyper Text Transfer Protocol (HTTP).
Application The internet has a wide range of applications in entertainment, education, research, navigation, advertising, and banking. The World Wide Web is utilized to access resources all across the world.

Evolution of the Internet

The internet has evolved from its origins as a military research project into a worldwide communication and information platform. It has transitioned from basic text-based communication to a multimedia-rich environment, enabling real-time collaboration, e-commerce, social networking, and more.

Internet Protocols

Internet protocols are standardised rules that govern how data is transmitted and received over the internet. Some key protocols include :

HTTP (Hypertext Transfer Protocol): Used for transferring web pages and resources.

HTTPS (Hypertext Transfer Protocol Secure): Encrypts data for secure communication.

TCP/IP (Transmission Control Protocol/Internet Protocol): The foundational protocol suite of the internet.

SMTP (Simple Mail Transfer Protocol): Used for sending emails.

FTP (File Transfer Protocol): Used for transferring files between computers.
